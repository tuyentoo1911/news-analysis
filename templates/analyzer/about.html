Giới thiệu về News Analyzer

Hệ thống phân tích tin tức thông minh sử dụng AI

Mục tiêu

News Analyzer được phát triển với mục tiêu giúp người dùng phân biệt tin tức thật và tin giả trong thời đại thông tin bùng nổ. Hệ thống sử dụng các thuật toán machine learning tiên tiến để phân tích nội dung tin tức và đưa ra đánh giá về độ tin cậy.

Tính năng chính

  • Phát hiện tin giả với độ chính xác cao
  • Phân tích nội dung bằng AI
  • Giao diện thân thiện, dễ sử dụng
  • Thống kê chi tiết về kết quả
  • Hỗ trợ tiếng Việt tối ưu
  • Cập nhật mô hình liên tục

Công nghệ

Hệ thống được xây dựng trên nền tảng công nghệ hiện đại:

Backend
  • Django Framework
  • Python Machine Learning
  • Natural Language Processing
  • SQLite Database
Frontend
  • Bootstrap 5
  • JavaScript ES6+
  • Chart.js
  • Font Awesome Icons
